# 象与骑象人 — 项目开发计划（plan.md）

> **总工期**：10 天 | **技术栈**：纯 HTML5 + CSS3 + JavaScript | **开发者**：无编程经验，AI 辅助开发

---

## 开发原则

1. **先基础后功能**：先做共享模块，再做业务模块
2. **先核心后附加**：每个模块先做核心逻辑，再做 UI 和附加功能
3. **先游戏后智能体**：游戏模块更直观，先完成获得信心
4. **每天可验证**：每天结束时要能在浏览器中看到进展
5. **细粒度拆分**：考虑到开发者无编程经验，每个任务足够具体
6. **预留缓冲**：Day 9-10 预留充足的测试和修复时间
