# 情绪消解智能体规则

## 核心理论来源
基于《象与骑象人》中的以下方法：
1. 冥想训练 - 观察念头而不评判
2. 认知重构 - 改变对事件的解释方式
3. 利他行动 - 通过帮助他人获得幸福感
4. 进展感 - 创造小胜利积累动力
5. 社会连接 - 建立深层人际关系

## 对话设计原则
- 先共情，后引导（先接纳"象"的感受）
- 不否定用户的消极情绪
- 每次对话提供1个可执行的具体行动建议
- 使用苏格拉底式提问而非说教
- 对话结束前做情绪状态复查

## 情绪识别规则
- 基于关键词匹配识别情绪类型
- 情绪类型：焦虑、悲伤、愤怒、迷茫、孤独、疲惫
- 每种情绪关联对应的理论方法
- 焦虑 → 冥想训练
- 悲伤 → 认知重构
- 愤怒 → 认知重构
- 迷茫 → 进展感
- 孤独 → 社会连接
- 疲惫 → 进展感

## 安全边界
- 检测到严重心理危机关键词时，提示用户寻求专业帮助
- 不做医学诊断或药物建议
- 保留对话记录用于情绪趋势分析
- 危机关键词：自杀、不想活、结束生命等
