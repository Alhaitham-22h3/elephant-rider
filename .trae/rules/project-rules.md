# 项目通用规则

## 项目概述
本项目基于积极心理学著作《象与骑象人》，开发两个产品：
1. 互动叙事游戏：玩家扮演"骑象人"，与代表情绪本能的"象"互动，通过选择推进故事
2. 情绪消解智能体：基于书中理论方法，帮助用户识别和转化消极情绪的AI对话助手

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript（原生，不使用框架）
- 所有代码必须能在浏览器中直接运行，无需构建工具
- 文件编码统一使用 UTF-8

## 项目目录结构
- .trae/rules/   → 规则文件
- .trae/skills/  → 技能文件
- docs/          → 项目文档（架构、规格、计划）
- game/          → 互动叙事游戏模块
  - game/css/    → 游戏样式
  - game/js/     → 游戏脚本
  - game/assets/ → 游戏资源
- agent/         → 情绪消解智能体模块
  - agent/css/   → 智能体样式
  - agent/js/    → 智能体脚本
- shared/        → 两个模块共享的资源
  - shared/css/  → 共享样式
  - shared/js/   → 共享脚本

## 代码风格
- 变量命名使用驼峰式(camelCase)
- 函数命名使用动词开头
- CSS类名使用短横线连接(kebab-case)
- 代码缩进使用2个空格

## 禁止事项
- 禁止使用alert/prompt/confirm，使用自定义UI组件替代
- 禁止引入任何需要npm安装的依赖
- 禁止硬编码敏感信息（API密钥等）
- 禁止使用已废弃的API

## 国际化
- 所有用户可见文本使用中文
- 代码注释使用中文
